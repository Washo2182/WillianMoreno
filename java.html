Lenguaje Java

Java es uno de los lenguajes más utilizados en el desarrollo empresarial, aplicaciones móviles (Android) y sistemas embebidos.

¿Por qué usar Java?

Es multiplataforma, orientado a objetos y tiene una enorme comunidad de desarrolladores.

Aplicaciones de Java

  • Desarrollo móvil (Android)
  • Sistemas empresariales
  • Juegos y simuladores
